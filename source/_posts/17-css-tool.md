作为WEB前端开发人员，你的工作可能很大一部分都在编写CSS代码，为了提高前端开发人员编写CSS代码的效率，编程文库从 网上搜集了17款可以提高你CSS代码效率的CSS工具，它们可以帮助你快速生成CSS菜单、动画影像、按钮、滑块、动画文本、CSS形 状代码。全文请看：17款提高编码效率的CSS工具。 不用介绍，软件开发界的“地球人”都知道CSS，因为它是最流行的一种样式设计语言。学习和使用CSS已经不再是一件棘手的事，因为在网上可以获得很多信息教程和CSS工具。这些工具有助于你创造出有用的、有创新的Web应用程序，同时节省更多开发步骤和时间。 文章中收集了加速编码的17款最好的CSS工具；以帮助开发人员和设计人员简化工作。利用它们可以生成CSS菜单、动画影像、按钮、滑块、动画文本、CSS形状代码、图层还有很多其他的。希望这个列表对你的开发工作有用。 **1.** **Topcoat : CSS for Clean and Fast Web Apps** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/00112.jpg "001") Topcoat是一个程序库，包含CSS类用于创建表格元素、按钮、复选框、滑块等。这个轻量级的工具可以让你设计的网页或者App看起来相当不错。 **2.** **PCSS : Shortcut Oriented Server Side CSS3 Preprocessor** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/0026.jpg "002") PCSS是一款由PHP驱动的CSS预处理器，它可以帮助开发者通过定义变量、类测试、默认单元和服务端浏览器细节，以达到快速编写CSS代码的目的。PCSS使用PHP5来实现服务端处理，因此正在运行的网站服务器上需要具备这个PCSS版本。 **3.** **Skeleton** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/0034.jpg "003") Skeleton是一款较小的CSS文件集合体，它可以帮助开发者快速创建适配任何尺寸的网站，无论是在17寸笔记本屏幕或iPhone上。Skeleton的构建基于三大规律：快速响应到手机，迅速启动和独一无二的风格。 **4.** **CSS Menu Maker** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/0042.jpg "004") 这个工具帮助用户轻松创建CSS下移菜单。CSS菜单编辑器提供网络专业人员和工具，帮助开发人员构建个性化的、跨浏览器相兼容的CSS菜单，同时还为所有的CSS菜单和设备用户提供源代码，以便下载和调整代码。 **5.** **Sencha Animator** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/0052.jpg "005") 这是一个桌面应用程序，允许用户创建专门用在触摸屏移动设备和Webkit浏览器上的CSS3动画。 Sencha Animator还可以协助用户创建动画文本、图片、倾斜按钮和内嵌分析。 **6.** **CSS Form Code Generator** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/006.jpg "006") CSS Form Code Generator可以制作布局好看的表格，也可以制作出布局简单的色彩表。这个“Code Maker”生成的CSS布局代码也可以为这些表格添枝加叶。 **7.** **PrefixmyCSS** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/007.jpg "007") PrefixmyCss可以简简单单的为CSS3代码添加前缀。用户必须复制CSS代码到窗口才能添加前缀，供应商已提前在代码上添加了前缀，用户可安全放心的替换旧的代码。 **8.** **Sky CSS Tool** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/008.jpg "008") Sky CSS Tool允许你创建CSS类时几乎不需要使用手写代码。用户可能需要与JavaScript相兼容的浏览器来确保正常运行。 **9.** **Spritemapper** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/009.jpg "009") Spritemapper是一款可以合并多个图片的应用程序，为相应的片段生成CSS精确位置。此款工具可以帮助开发者优化可用空间和加载时间。 **10.** **CSS Compressor** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/010.jpg "010") CSS Compressor通过压缩CSS来提高加载速度，还可以节约带宽。依据你对CSS的压缩程度和代码的易读性，它提供了三个级别的压缩。 **11.** **Patternify** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/11.jpg "11") Patternify可以帮助你生成漂亮的CSS模板。 **12.** **CSS Text Shadow** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/12.jpg "12") CSS Text Shadow帮助使用者生成漂亮的文字阴影。 **13.** **CSS3 Pie** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/13.jpg "13") CSS3 Pie在IE 6-9下可以渲染出最有用的CSS装饰功能。 **14.** **The Web Font Combinator** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/14.jpg "14") 该工具旨在浏览器下快速预览字体，无需一遍遍的刷新。近些年出版的书也都让标题字体紧挨着正文字体，这也是一种网络再创作的尝试。 **15.** **3D Transforms** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/16.jpg "16") CSS 3D变换器是一款在线工具，用户使用它可以执行多种级别的3D转换。目前此工具还处在实验模式下，在使用的时候，需要在浏览器下制定浏览器前缀。 **16.** **Quick Form Builder** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/161.jpg "16") Quick Form Builder有助于便捷地创建CSS表单。 **17.** **Layer Style** ![](http://wiki.itivy.com/wp-content/uploads/2013/06/17.jpg "17") 这是一款HTML5 App，用一种直观的方式创建CSS3。其内置的Colorpicker允许用户选择各种颜色，还便于用户拖放图片到页面，使用它们作为背景或挑选他们喜欢的颜色。
